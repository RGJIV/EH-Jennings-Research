# Directory Scans Archive

This folder is intended to hold the original Pittsburgh city directory page images that support the entries in `significant-addresses.md` and `sources.md`.

## Recommended structure

```
directory-scans/
  1900/
  1905/
  1907/
  1910/
  1914/
  1915/
```

## Current status (23 July 2026)

The original screenshots were supplied in the research chat and have been fully transcribed into the address and source tables.  
They have **not** yet been bulk-uploaded as binary files because of size limits in the current connector workflow.

**Action for permanent archive**  
Download the screenshots from the chat history and place them in the year folders above. Suggested filenames:

- `1900-jennings-personal.jpg`
- `1900-jennings-business.jpg`
- `1905-jennings-section.png`
- `1905-pure-oil-producing.png`
- `1905-colonial-trust.png`
- `1907-colonial-trust.png`
- `1907-columbia-national-bank.png`
- `1907-oil-companies.png`
- `1910-jennings-section.png`
- `1910-oil-companies.png`
- etc.

Once the images are in place, each row in `significant-addresses.md` can be linked to the corresponding scan file.

## Why this matters

Keeping the original page images preserves the primary evidence and protects against transcription error. All textual claims in the project should ultimately be traceable to one of these scans or to a loc.gov / court record.
